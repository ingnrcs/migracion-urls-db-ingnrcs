Script - Migración de WordPress

Esta herramienta te permite migrar las URLs en tu sitio de WordPress de forma sencilla.

// required disabled for local db tests

Pasos para usar la herramienta:

  1. Ingresa la URL antigua y la nueva en los campos correspondientes.
  2. Proporciona la información de conexión a la base de datos. ("esta información la encontrarás en el archivo wp-config de tu ruta raíz de WordPress en líneas con el siguiente contenido o similar: define( 'DB_NAME', 'NOMBRE_BASE_DE_DATOS' ), define( 'DB_USER', 'NOMBRE_USUARIO' ), define( 'DB_PASSWORD', 'PASSWORD' )").
  3. Especifica el prefijo de las tablas de tu sitio ("esta información también la encontrarás en el wp-config.php en la variable llamada: $table_prefix = ''").
  4. Haz clic en "Migrar" para comenzar el proceso de migración.
  5. Una vez completada la migración, recibirás estadísticas de cambios realizados.
  6. IMPORTANTE: para evitar problemas de seguridad, no olvides eliminar el script después de reemplazar las URLs de tu sitio.
